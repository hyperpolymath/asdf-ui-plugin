== Overview

This repository will contain an https://asdf-vm.com/[asdf] plugin for managing UI tooling versions.

== Current State

This repository currently contains infrastructure configuration only:

* Multi-forge mirroring (GitHub, GitLab, Codeberg, Bitbucket)
* Instant sync propagation for releases
* Development guidelines (CLAUDE.md)

No plugin source code has been implemented yet.
